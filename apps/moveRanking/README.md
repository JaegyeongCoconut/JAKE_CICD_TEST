# KOKKOK RANKING WEB

KOKKOK RANKING WEB is a mobile web application that ranks RANKING Heroes.

## How to start

### Local Development

```bash
pnpm install
pnpm start
```

### Local Production

```bash
pnpm install
pnpm run start-prod
```

### Deploy Development

```bash
pnpm install --production
pnpm run build
```

### Deploy Production

```bash
pnpm install --production
pnpm run build-prod
```
