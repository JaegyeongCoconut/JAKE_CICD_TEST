# KOKKOK CAR INSPECTION

KOKKOK CAR INSPECTION is a web application that manages vehicle inspections.

## How to start

### Local Development

```bash
pnpm install
pnpm run dev
```

### Local Production

```bash
pnpm install
pnpm run start-prod
```

### Deploy Development

```bash
pnpm install --production
pnpm run build
```

### Deply Production

```bash
pnpm install --production
pnpm run build-prod
```
